Player-facing disclosures for a North America-ready arcade game prototype.

Children & Family Play

Players under 13 should use Kids Mode or play with parent or guardian permission. Kids Mode disables coin pack checkout and uses a local-only profile.

Accessibility

Accessibility settings include high contrast, reduced motion, and larger controls. Keyboard, touch, and pointer input are supported.

Data Processing

This build stores pilot profile, country, coins, upgrades, purchases, achievements, and ranking data in localStorage on this device. Checkout uses the payment provider script only after the player confirms checkout details.

Player Control

Players can switch profiles by email, use Guest Access, or clear local game data from this device.
